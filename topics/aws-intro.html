What is AWS?

Amazon Web Services (AWS) is a cloud computing platform providing scalable computing, storage, and networking.

💡 AWS follows a Pay-as-you-go pricing model.

Cloud Service Models

AWS provides three major cloud models:

IaaS  → Infrastructure as a Service
PaaS  → Platform as a Service
SaaS  → Software as a Service

Benefits of AWS

✔ Scalability

✔ High Availability

✔ Security

✔ Global Infrastructure
