Section 1
Strategy & Big Data
QUESTION 01 · SELECT THREE

Which aspects comprise the three-pronged strategy?

Choose exactly three options.


✅ The AWS three-pronged data strategy is Modernize (move workloads to cloud), Unify (centralize data in a data lake), and Innovate (build analytics, AI, and ML solutions on top).
QUESTION 02

Which scenario describes a challenge to velocity?

✅ Velocity refers to the speed of data processing. The delay in returning recommendations during high traffic is a data processing speed problem — a classic velocity challenge.
QUESTION 03 · SELECT TWO

Which options are part of the five Vs of big data?

Choose exactly two options.


✅ The Five Vs of Big Data are Volume, Velocity, Variety, Veracity, and Value. Veracity = data accuracy & reliability. Value = useful insights generated from data.
QUESTION 04

Which option is NOT a pillar of the AWS Well-Architected Framework?

✅ The six pillars are Operational Excellence, Security, Reliability, Performance Efficiency, Cost Optimization, and Sustainability. Disaster Recovery is a concept within Reliability, not a standalone pillar.
Section 2
AWS Services & Ingestion
QUESTION 05

An air conditioning company monitors airflow through hospital ducts. Which AWS service would be well-suited to consume the streaming IoT data?

✅ Amazon Data Firehose collects, processes, and delivers real-time streaming data (such as IoT sensor readings) to destinations like S3, Redshift, or OpenSearch.
QUESTION 06

Which option is NOT a design principle for data security?

✅ The statement is incorrect because AWS permissions can also be managed using IAM users, groups, and policies — not only roles. Restricting to roles alone is not a security principle.
QUESTION 07

A system administrator wants to create alarms that send notifications when EC2 CPU utilization thresholds are breached. Which AWS service best meets their needs?

✅ Amazon CloudWatch monitors AWS resource metrics like EC2 CPU utilization and can trigger alarms that send notifications (e.g., via SNS) when thresholds are exceeded.
QUESTION 08

Which AWS service is designed to ingest data from file systems?

✅ AWS DataSync transfers and ingests data from on-premises file systems (NFS/SMB) to AWS storage services like Amazon S3, EFS, or FSx.
Section 3
Data Wrangling & Structuring
QUESTION 09

What is the correct order of tasks in data structuring?

✅ Correct workflow: 1) Organize storage (decide where data lives) → 2) Parse (read raw data) → 3) Map fields (match to schema) → 4) Manage file size (optimize for processing).
QUESTION 10

Which term is NOT a step in data wrangling?

✅ Data wrangling steps are Discovery, Structuring, Cleaning, Enriching, and Validating. Loading belongs to the data ingestion / ETL process, not wrangling.
QUESTION 11 · SELECT TWO

Which AWS services can be used to monitor and troubleshoot an AWS Glue job?

Choose exactly two options.


✅ Amazon CloudWatch monitors Glue job logs, metrics, and performance. AWS CloudTrail tracks API calls and activity history — both are essential for troubleshooting Glue jobs.
Section 4
Databases & Storage
QUESTION 12

A company wants to build a website to show leaderboard scores for a high-traffic computer game. Which database would be BEST suited for this purpose?

✅ Amazon DynamoDB is a fully managed NoSQL database with high throughput and low latency — ideal for real-time applications like gaming leaderboards. 🎮
QUESTION 13

Which service is MOST commonly associated with data warehousing?

✅ Amazon Redshift is AWS's fully managed data warehouse service used for large-scale analytics and SQL queries on structured data. 📊
QUESTION 14

Which service would a data engineer use to encrypt the data in their data lake?

✅ AWS KMS creates and manages encryption keys used to encrypt data stored in services like Amazon S3 data lakes. 🔐
Section 5
Amazon EMR
QUESTION 15 · SELECT THREE

Which options are Amazon EMR node types?

Choose exactly three options.


✅ Amazon EMR has three node types: Main (manages the cluster), Core (runs tasks + stores data in HDFS), and Task (runs processing tasks, no data storage).
QUESTION 16

How many layers does the Amazon EMR service architecture consist of?

✅ Amazon EMR has four architecture layers: 1) Storage (S3, HDFS), 2) Cluster resource management (YARN), 3) Data processing (Spark, Hive, etc.), 4) Application layer (analytics tools).
QUESTION 17

Which tool or service is NOT used to handle near real-time data?

✅ Apache Pig is primarily used for batch data processing in Hadoop. Apache Flink, Apache Storm, and AWS Lambda are all capable of processing real-time or near real-time data.
Section 6
Machine Learning & AI
QUESTION 18

Which step is NOT part of framing a typical machine learning (ML) problem?

✅ Using a 20% test set to define schema is a data preparation step, not part of problem framing. Framing involves checking if ML fits, defining metrics, and planning a simple starting model.
QUESTION 19

How does machine learning (ML) differ from traditional data analysis?

✅ ML learns patterns from data examples, while traditional analysis relies on manually written rules or programming logic to answer questions.
QUESTION 20

A developer wants to use a generative AI service embedded in an IDE to help generate code. Which AWS service fits?

✅ Amazon Q Developer is a generative AI coding assistant that integrates with IDEs (VS Code, JetBrains, etc.) to generate code, suggest fixes, and improve developer productivity. 💻
QUESTION 21

What percentage of a machine learning (ML) dataset should be allocated to training?

✅ The standard ML split is 70–80% training and 20–30% testing/validation. This allows the model to learn from most of the data and be evaluated on unseen data.
Section 7
Analytics, BI & DevOps
QUESTION 22

Which AWS service provides cloud-scale business intelligence to help deliver insights that are easy to understand?

✅ Amazon QuickSight is AWS's Business Intelligence (BI) service for creating dashboards, visualizations, and reports to understand data insights. 📊
QUESTION 23

Users with different roles at a video game company use AWS tools within different stages of the analytics pipeline. Which role would you associate with exploring and analyzing player data?

✅ A Data Analyst explores, queries, and analyzes data to find insights — such as player behavior patterns and game performance metrics. 🎮📊
QUESTION 24 · SELECT THREE

What are the MOST common use cases for Amazon OpenSearch Service?

Choose exactly three options.


✅ Amazon OpenSearch Service is most commonly used for: Log analytics (system/application logs), real-time application monitoring, and clickstream analysis (tracking user behavior on websites/apps).
QUESTION 25

What is the main difference between continuous delivery and continuous deployment?

✅ Continuous Delivery — code is ready for deployment but typically requires manual approval. Continuous Deployment — code is automatically deployed to production without manual intervention. 🚀
